Python2 Unicode
=============
```python
# -*- encoding: utf-8 -*-
from typograf import RemoteTypograf
rt = RemoteTypograf(p=True, br=True) # UTF-8
print rt.process_text(u'"Вы все еще кое-как верстаете в "Ворде"? - Тогда мы идем к вам!"')
```
